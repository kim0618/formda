## 9단계. 유지 7편에 링크 삽입 (글당 1개, 본문 하단)

> UTM `utm_campaign=legacy`로 신규 글(f{N})과 분리 측정. 아래 문장 그대로 본문 맨 아래에 붙이면 됨.

**① 1인 사업자 경비 처리 - 세금 줄이는 증빙 챙기는 법** → 폼다 영수증
```
증빙용 영수증이 필요하면 폼다에서 바로 만들 수 있어요.
https://formda.kr/tools/receipt?utm_source=naverblog_formda&utm_campaign=legacy&utm_content=bottom
```

**② 간이과세자 일반과세자 차이 - 1인 사업자 뭐가 유리할까** → 제이퍼 부가세 계산기
```
내 매출로 부가세가 얼마 나오는지는 제이퍼 계산기에서 확인해 보세요.
https://www.jptcalc.kr/calc/tax/vat/?utm_source=naverblog_formda&utm_campaign=legacy
```

**③ 1인 사업자 부가세 신고 - 7월 놓치면 가산세 나오는 일정** → 폼다 청구서
```
거래처에 보낼 청구서는 폼다에서 부가세까지 자동 계산해서 만들 수 있어요.
https://formda.kr/tools/invoice?utm_source=naverblog_formda&utm_campaign=legacy&utm_content=bottom
```

**④ 1인 사업자 부가세 신고 - 7월 신고 미리 준비하는 법** → 폼다 거래명세서
```
신고 전 거래 내역을 정리할 때 쓰는 거래명세서는 여기서 바로 만들 수 있어요.
https://formda.kr/tools/statement?utm_source=naverblog_formda&utm_campaign=legacy&utm_content=bottom
```

**⑤ 1인 사업자 부모님 부양가족 공제 - 어버이날 챙기는 절세** → 제이퍼 종소세
```
공제를 반영하면 세금이 얼마나 줄어드는지 제이퍼 계산기로 확인해 보세요.
https://www.jptcalc.kr/calc/tax/income-tax/?utm_source=naverblog_formda&utm_campaign=legacy
```
※ 부양가족 공제 전용 계산기는 없어서 종소세 계산기로 연결(실확인)

**⑥ 종소세 환급 받는 5가지 방법 - 1인 사업자가 놓치는 공제** → 제이퍼 종소세
```
환급액이 궁금하면 제이퍼 종합소득세 계산기에서 확인해 보세요.
https://www.jptcalc.kr/calc/tax/income-tax/?utm_source=naverblog_formda&utm_campaign=legacy
```

**⑦ 1인 사업자 종합소득세 신고 - 처음 신고 체크리스트** → 제이퍼 프리랜서 소득
