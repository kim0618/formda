텍스트 정렬

여러 줄 텍스트를 정렬하고 중복·빈 줄 제거, 공백 정리, 줄 번호 매기기를 실시간으로 처리합니다.

🔒 입력 정보는 서버에 저장되지 않고 브라우저에서만 처리됩니다.

텍스트 정렬 사용 가이드

텍스트 정렬은 여러 줄로 된 텍스트를 가나다순·ABC순으로 정렬하고, 중복된 줄과 빈 줄을 제거하거나 줄마다 번호를 매기는 등 줄 단위 정리를 한 번에 해 주는 도구입니다. 명단을 정렬하거나 키워드 목록에서 중복을 빼는 작업을 엑셀 없이 브라우저에서 바로 처리할 수 있고, 옵션을 켜고 끄는 즉시 결과가 갱신됩니다. 입력 내용은 서버에 저장되지 않습니다.

한 번에 처리할 수 있는 정리 작업

여러 옵션을 함께 적용할 수 있어, 흩어진 목록을 깔끔한 결과로 한 번에 정리합니다.

  • 정렬 - 가나다·ABC 오름차순 또는 내림차순으로 줄을 정렬합니다.
  • 중복 줄 제거 - 똑같은 줄이 여러 번 있으면 하나만 남깁니다.
  • 빈 줄 제거 - 내용 없는 줄을 없애 목록을 압축합니다.
  • 앞뒤 공백 제거·줄 번호 - 줄 양끝 공백을 정리하고 1, 2, 3 번호를 매깁니다.

이런 작업에 편합니다

  • 참석자·고객 명단을 가나다순으로 정렬
  • 키워드·태그 목록에서 중복 제거
  • 복사한 목록의 빈 줄과 공백 정리
  • 번호가 필요한 목록에 줄 번호 매기기

옵션을 조합한 정리 예시

여러 옵션을 동시에 켜면 흩어진 목록을 한 번에 깔끔하게 만들 수 있습니다. 명단 정리를 예로 들어 보겠습니다.

단계결과
입력 김철수 / 이영희 / (빈 줄) / 김철수 / 박민수
앞뒤 공백 제거 + 빈 줄 제거김철수 / 이영희 / 김철수 / 박민수
중복 줄 제거김철수 / 이영희 / 박민수
오름차순 정렬김철수 / 박민수 / 이영희

네 가지 옵션을 한꺼번에 켜면 위 과정을 한 번에 처리해 결과가 바로 나옵니다.

한글·영문·숫자 정렬 기준

  • 한글 - 가나다순으로 정렬됩니다.
  • 영문 - 대소문자를 구분하지 않고 사전식(알파벳순)으로 정렬됩니다.
  • 숫자 - 문자로 비교되어 1, 10, 2, 3 순서가 될 수 있습니다. 순서를 맞추려면 01, 02처럼 자릿수를 맞추세요.

자주 묻는 질문

Q. 숫자가 섞인 줄은 어떻게 정렬되나요?

정렬은 글자 기준이라 숫자가 섞이면 1, 10, 2, 20, 3처럼 사람이 기대하는 순서와 다르게 정렬될 수 있습니다. 이는 숫자를 하나의 수가 아니라 문자 그대로 앞에서부터 비교하기 때문입니다. 순서를 제대로 맞추려면 1, 2, 3을 01, 02, 03처럼 자릿수를 맞춰 입력하면 됩니다. 한글과 영문이 섞인 경우에는 각각 묶여 정렬됩니다.

Q. 정렬은 어떤 기준인가요?

한국어 정렬(localeCompare) 기준으로, 한글은 가나다순, 영문은 알파벳순으로 정렬됩니다. 오름차순과 내림차순을 선택할 수 있습니다.

Q. 여러 옵션을 같이 쓸 수 있나요?

네. 공백 제거, 빈 줄 제거, 중복 제거, 정렬, 줄 번호 매기기를 함께 적용할 수 있고, 결과는 즉시 갱신됩니다.

Q. 입력한 내용이 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 입력 내용을 서버로 전송하거나 저장하지 않습니다.
