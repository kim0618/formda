PDF 페이지 회전

옆으로 눕거나 거꾸로 스캔된 PDF를 90도·180도로 돌려 바로 내려받을 수 있습니다.

🔒 입력 정보는 서버에 저장되지 않고 브라우저에서만 처리됩니다.

PDF 페이지 회전 사용 가이드

PDF 페이지 회전은 스캐너나 휴대폰으로 문서를 찍을 때 방향이 잘못되어 옆으로 눕거나 거꾸로 나온 PDF를 바로 세워 주는 도구입니다. 전체 페이지를 한 번에 돌릴 수도 있고, 양면 스캔에서 뒷면만 거꾸로 나온 경우처럼 홀수 쪽이나 짝수 쪽만 골라 돌릴 수도 있습니다. 처리는 모두 사용자 브라우저 안에서 이루어져 파일이 서버로 전송되지 않습니다.

PDF 회전, 이럴 때 씁니다

  • 스캔 방향 오류 - 스캐너에 문서를 옆으로 넣어 페이지가 90도 눕게 나왔을 때.
  • 거꾸로 찍힌 사진 변환본 - 휴대폰 사진을 PDF로 만들었는데 180도 뒤집혀 있을 때.
  • 양면 스캔 뒷면 오류 - 앞면은 정상인데 뒷면(짝수 쪽)만 거꾸로 나왔을 때.

전체 회전과 홀수·짝수 선택 회전

범위이럴 때
전체 페이지문서 전체가 같은 방향으로 잘못 스캔됐을 때
홀수 쪽만앞면(1, 3, 5쪽 등)만 방향이 틀렸을 때
짝수 쪽만뒷면(2, 4, 6쪽 등)만 방향이 틀렸을 때

양면 스캐너로 문서를 넣으면 뒷면만 반대 방향으로 인식되는 경우가 흔한데, 이때 전체를 돌리면 앞면까지 잘못되므로 짝수 쪽만 선택해 돌려야 합니다.

회전하는 법

  • 1. 돌릴 PDF를 끌어다 놓거나 클릭해 선택합니다.
  • 2. 회전 방향(90도 시계·반시계, 180도)과 적용 범위(전체·홀수·짝수)를 고릅니다.
  • 3. 회전하기를 누르면 방향이 바로잡힌 PDF가 곧바로 내려받아집니다.

자주 묻는 질문

Q. 페이지마다 다른 방향으로 회전할 수 있나요?

전체·홀수 쪽·짝수 쪽 단위로 선택해 회전할 수 있습니다. 페이지 한 장 한 장을 서로 다른 각도로 돌리는 기능은 없으므로, 방향이 제각각이라면 PDF 분할로 나눠 페이지별로 따로 돌린 뒤 PDF 합치기로 다시 합치는 방법을 쓸 수 있습니다.

Q. 90도로 돌렸는데 반대 방향이면 어떻게 하나요?

시계방향과 반시계방향을 반대로 다시 선택해 한 번 더 돌리면 됩니다. 예를 들어 90도 시계방향으로 돌렸는데 반대였다면, 그 결과 파일을 다시 올려 90도 반시계방향(또는 시계방향으로 3번)으로 돌리면 원래 의도한 방향이 됩니다.

Q. 회전하면 화질이 떨어지나요?

아니요. 페이지 내용을 다시 그리거나 압축하지 않고, PDF 안에 저장된 회전 값만 바꾸는 방식이라 글자와 이미지가 원본 그대로 유지됩니다.

Q. 양면 스캔 뒷면만 거꾸로 나왔는데 어떻게 하나요?

적용 범위에서 짝수 쪽만(또는 홀수 쪽만)을 선택하고 180도 회전을 고르면, 뒷면(짝수 쪽)만 방향이 바로잡히고 앞면(홀수 쪽)은 그대로 유지됩니다.

Q. 파일이 서버에 올라가나요?

아니요. PDF 회전은 모두 사용자 브라우저 안에서 처리되며, 파일을 서버로 전송하거나 저장하지 않습니다. 방향이 바로잡힌 결과 PDF만 사용자 기기에 내려받아집니다.
