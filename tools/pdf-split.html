PDF 분할

PDF에서 원하는 페이지만 뽑아 하나로 만들거나, 모든 페이지를 낱장 PDF로 분리해 내려받을 수 있습니다.

🔒 입력 정보는 서버에 저장되지 않고 브라우저에서만 처리됩니다.

PDF 분할 사용 가이드

PDF 분할은 한 개의 PDF에서 필요한 페이지만 뽑아내거나, 모든 페이지를 한 장씩 개별 파일로 나눠 주는 도구입니다. 여러 장짜리 계약서에서 특정 장만 제출해야 할 때, 스캔한 문서를 페이지별로 나눠 관리할 때 씁니다. 페이지 범위를 지정해 하나의 PDF로 추출하거나, 낱장 분리로 모든 페이지를 개별 PDF로 만들어 ZIP으로 받을 수 있습니다. 처리는 모두 사용자 브라우저 안에서 이루어져 파일이 서버로 전송되지 않습니다.

PDF 분할, 이럴 때 씁니다

  • 필요한 장만 제출 - 여러 장 문서에서 특정 페이지만 뽑아 낼 때.
  • 용량 줄이기 - 큰 PDF에서 필요한 부분만 남겨 가볍게 만들 때.
  • 페이지별 관리 - 스캔본을 한 장씩 나눠 따로 보관·전달할 때.

두 가지 분할 방식

방식결과이럴 때
페이지 추출지정한 페이지만 모은 PDF 1개3~5쪽만 필요할 때
낱장 분리모든 페이지를 개별 PDF로 (ZIP)페이지마다 따로 써야 할 때

페이지 추출은 "1-3, 5, 8-10"처럼 쉼표로 여러 범위를 지정할 수 있고, 지정한 순서·개수만큼만 담깁니다.

분할하는 법

  • 1. 나눌 PDF를 끌어다 놓거나 클릭해 선택합니다(총 페이지 수가 표시됩니다).
  • 2. 페이지 추출을 고르고 원하는 페이지 범위를 입력하거나, 낱장 분리를 고릅니다.
  • 3. 분할하기를 누르면 추출본(PDF) 또는 낱장 모음(ZIP)이 바로 내려받아집니다.

자주 묻는 질문

Q. 특정 페이지만 뽑아낼 수 있나요?

네. 페이지 추출을 선택하고 "1-3, 5, 8-10"처럼 원하는 페이지와 범위를 쉼표로 입력하면 그 페이지들만 모은 하나의 PDF가 만들어집니다. 존재하지 않는 페이지 번호는 자동으로 무시되며, 지정한 순서대로 담깁니다. 연속 구간은 하이픈(-)으로, 낱개 페이지는 숫자로 적으면 됩니다.

Q. 낱장 분리는 어떻게 받아지나요?

낱장 분리를 선택하면 PDF의 모든 페이지가 각각 한 장짜리 개별 PDF로 나뉘고, 이 파일들을 하나의 ZIP 파일로 묶어 내려받습니다. 파일 이름은 page-01.pdf, page-02.pdf처럼 페이지 순서대로 붙습니다. ZIP을 풀면 페이지별 PDF를 따로 쓸 수 있습니다.

Q. 파일이 서버에 올라가나요?

아니요. PDF 분할은 모두 사용자 브라우저 안에서 처리되며, 파일을 서버로 전송하거나 저장하지 않습니다. 만들어진 추출본이나 ZIP만 사용자 기기에 내려받아집니다. 계약서·개인정보가 담긴 문서도 외부로 나가지 않아 안심하고 나눌 수 있습니다.

Q. 분할하면 화질이나 내용이 바뀌나요?

아니요. 원본 페이지를 그대로 복사해 새 PDF로 옮기므로 글자·이미지·서식이 원본 그대로 유지됩니다. 다시 압축하거나 변환하지 않아 화질 저하가 없습니다.

Q. 암호가 걸린 PDF도 나눌 수 있나요?

열람 암호가 걸려 내용을 열 수 없는 PDF는 분할할 수 없으며 안내가 표시됩니다. 암호를 알고 있다면 PDF 뷰어에서 암호를 해제해 저장한 뒤 올리면 됩니다. 페이지가 매우 많은 PDF는 낱장 분리 시 처리에 시간이 조금 걸릴 수 있습니다.
