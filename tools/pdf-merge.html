PDF 합치기

여러 PDF 파일을 끌어다 놓으면 원하는 순서대로 하나의 PDF로 합쳐 바로 내려받을 수 있습니다.

🔒 입력 정보는 서버에 저장되지 않고 브라우저에서만 처리됩니다.

PDF 합치기 사용 가이드

PDF 합치기는 여러 개의 PDF 파일을 원하는 순서대로 이어 붙여 하나의 PDF로 만들어 주는 도구입니다. 계약서와 첨부서류, 여러 번에 나눠 스캔한 문서, 각각 만든 자료를 한 파일로 묶어 제출하거나 보관할 때 씁니다. PDF를 끌어다 놓고 순서만 정하면 바로 합쳐지며, 페이지 수나 파일 개수 제한이 없습니다. 합치는 과정은 모두 사용자 브라우저 안에서 처리되어 파일이 서버로 전송되지 않습니다.

PDF 합치기, 이럴 때 씁니다

  • 서류 제출 - 계약서, 첨부서류, 증빙을 하나의 PDF로 묶어 이메일·접수 사이트에 낼 때.
  • 스캔 문서 병합 - 여러 번에 나눠 스캔한 파일을 한 문서로 합칠 때.
  • 자료 취합 - 각자 만든 보고서·발표자료를 하나로 모을 때.

대부분의 접수 시스템은 파일 여러 개보다 PDF 한 개를 요구합니다. 순서만 맞춰 합치면 페이지가 뒤섞이지 않고 관리가 편합니다.

합치는 법

  • 1. 합칠 PDF들을 상자에 끌어다 놓거나 클릭해 여러 개를 한 번에 선택합니다.
  • 2. 목록에서 ▲ ▼ 버튼으로 순서를 바꾸고, ✕로 필요 없는 파일을 뺍니다.
  • 3. PDF 합치기를 누르면 추가한 순서대로 하나의 PDF로 합쳐져 바로 내려받아집니다.

합치기 전에 알아 두면 좋은 것

상황이렇게
순서가 중요할 때▲▼로 순서를 먼저 맞춘 뒤 합칩니다(추가한 순서 = 결과 순서).
특정 페이지만 빼고 싶을 때PDF 분할로 먼저 필요한 페이지만 추출한 뒤 합칩니다.
암호가 걸린 PDF암호로 열람이 막힌 파일은 합칠 수 없어 제외됩니다. 암호를 푼 뒤 올리세요.
용량이 큰 PDF페이지가 아주 많으면 처리에 시간이 걸릴 수 있습니다. 잠시 기다리면 완료됩니다.

자주 묻는 질문

Q. 합칠 수 있는 PDF 개수에 제한이 있나요?

개수 제한은 없습니다. 필요한 만큼 여러 PDF를 한 번에 추가해 하나로 합칠 수 있으며, 각 파일의 페이지 수도 제한하지 않습니다. 다만 전체 페이지가 매우 많거나 용량이 큰 경우 브라우저에서 처리하는 데 시간이 걸릴 수 있습니다. 순서는 추가한 차례대로 정해지며 ▲▼ 버튼으로 언제든 바꿀 수 있습니다.

Q. 파일이 서버에 올라가나요?

아니요. PDF 합치기는 모두 사용자 브라우저 안에서 처리되며, 파일을 서버로 전송하거나 저장하지 않습니다. 합쳐진 PDF만 사용자 기기에 내려받아집니다. 계약서·개인정보가 담긴 민감한 문서도 외부로 나가지 않아 안심하고 합칠 수 있습니다.

Q. 합치면 원본 화질이나 내용이 바뀌나요?

아니요. 각 PDF의 페이지를 그대로 복사해 이어 붙이므로 글자·이미지·서식이 원본 그대로 유지됩니다. 다시 압축하거나 변환하지 않아 화질 저하가 없습니다. 페이지 순서만 여러분이 정한 대로 배치됩니다.

Q. 암호가 걸린 PDF도 합칠 수 있나요?

열람 암호가 걸려 내용을 열 수 없는 PDF는 합칠 수 없어 자동으로 제외되고 안내가 표시됩니다. 암호를 알고 있다면 PDF 뷰어에서 암호를 해제해 저장한 뒤 올리면 됩니다. 편집·인쇄만 제한된 일부 PDF는 합쳐질 수 있습니다.

Q. 합친 뒤 특정 페이지를 빼거나 순서를 바꾸려면요?

합치기 전에 목록에서 ✕로 파일을 빼거나 ▲▼로 순서를 조정하면 됩니다. 한 파일 안의 특정 페이지만 빼야 한다면 PDF 분할 도구로 필요한 페이지만 먼저 추출한 뒤, PDF 합치기로 다시 합치면 원하는 구성으로 만들 수 있습니다.
