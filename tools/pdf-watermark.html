PDF 워터마크

PDF에 원하는 문구를 워터마크로 반복해서 찍거나 가운데에 한 번 넣어 바로 내려받을 수 있습니다.

🔒 입력 정보는 서버에 저장되지 않고 브라우저에서만 처리됩니다.

PDF 워터마크 사용 가이드

PDF 워터마크는 문서 위에 "대외비", "초안" 같은 문구를 반투명하게 겹쳐 찍어, 문서의 상태를 표시하거나 무단 배포·유출을 막고 싶을 때 쓰는 도구입니다. 원하는 문구를 입력하면 대각선으로 회전한 글자가 반투명하게 페이지 전체에 반복되거나 가운데에 한 번 찍히며, 진하기도 세 단계로 조절할 수 있습니다. 처리는 모두 사용자 브라우저 안에서 이루어져 파일이 서버로 전송되지 않습니다.

PDF 워터마크, 이럴 때 씁니다

  • 대외비·기밀 표시 - 내부 문서가 외부로 유출됐을 때 출처와 상태를 알리고 싶을 때.
  • 초안·견본 구분 - 확정본이 아닌 검토용·샘플 문서임을 표시할 때.
  • 배포처 구분 - 같은 문서를 여러 곳에 배포하며 구분 문구를 남기고 싶을 때.

반복(타일)과 가운데 배치, 무엇이 다를까

배치결과이럴 때
반복(타일)페이지 전체에 문구가 여러 번 반복일부를 오려도 워터마크가 남아야 할 때
가운데 1개페이지 가운데에 큼직하게 한 번본문을 가리지 않고 깔끔하게 표시할 때

내용을 캡처하거나 일부만 잘라가도 표시가 남아야 한다면 반복(타일)을, 문서를 읽기 편하게 유지하면서 상태만 표시하고 싶다면 가운데 배치를 고르세요.

워터마크 넣는 법

  • 1. 워터마크를 찍을 PDF를 끌어다 놓거나 클릭해 선택합니다.
  • 2. 넣을 문구를 입력하고, 반복 또는 가운데 배치와 진하기를 고릅니다.
  • 3. 워터마크 넣기를 누르면 모든 페이지에 문구가 찍힌 PDF가 바로 내려받아집니다.

자주 묻는 질문

Q. 한글 문구도 워터마크로 넣을 수 있나요?

네. 입력한 문구를 브라우저에서 이미지로 그린 뒤 PDF에 찍는 방식이라 한글·영문·숫자 모두 문제없이 들어갑니다. "대외비", "초안", "견본" 등 원하는 문구를 자유롭게 입력하면 됩니다.

Q. 워터마크를 넣으면 원본 내용이 지워지나요?

아니요. 기존 페이지 내용은 그대로 두고 그 위에 반투명한 문구를 겹쳐 찍는 방식입니다. 진하기를 연하게로 선택하면 아래 글자를 읽는 데 크게 방해되지 않습니다.

Q. 나중에 워터마크만 지울 수 있나요?

폼다에서 만든 워터마크는 페이지 내용에 겹쳐 그려진 이미지라 다시 깨끗하게 지우기는 어렵습니다. 워터마크가 필요 없는 최종본은 원본 파일을 따로 보관해 두고, 배포용에만 워터마크를 넣는 방식을 권장합니다.

Q. 페이지마다 다른 문구를 넣을 수 있나요?

현재는 한 번에 같은 문구가 모든 페이지에 동일하게 들어갑니다. 페이지별로 다른 문구가 필요하다면 PDF 분할로 나눈 뒤 각각 워터마크를 넣고 다시 PDF 합치기로 합치면 됩니다.

Q. 파일이 서버에 올라가나요?

아니요. PDF 워터마크는 모두 사용자 브라우저 안에서 처리되며, 파일을 서버로 전송하거나 저장하지 않습니다. 워터마크가 찍힌 결과 PDF만 사용자 기기에 내려받아집니다.
